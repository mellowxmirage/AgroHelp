# 🌾 Smart Agriculture AI System

## 📌 Overview
An AI-powered agriculture system that helps farmers make data-driven decisions.

## 🚀 Features
- Crop Yield Prediction
- Pest Risk Analysis
- Fertilizer Recommendation
- Water Requirement Prediction


## 🧠 Tech Stack
- Python, Pandas, NumPy
- Scikit-learn
- Streamlit
- FastAPI
- SQLite

## 📊 Models Used
- Classification (Crop Identification)

git clone https://github.com/mellowxmirage/AgroHelp
